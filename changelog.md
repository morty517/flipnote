v1.0:
First release! Full text editor for Flipper Zero.
- Open / Save / Save As with native file browser
- Virtual buffer for large files (up to 2000 lines, streams from SD card)
- Find with match navigation
- Find and Replace (streams through entire file)
- Copy and Paste line
- Delete Row / Clear All
- Goto Row with number input
- Horizontal scroll, row numbers stay fixed
- 8 zoom levels (0.25x to 2.00x)
- Row numbers toggle
- First Row / Last Row instant jump
- Extended symbol keyboard on Momentum firmware
